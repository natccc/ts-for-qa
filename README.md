# 📘 TypeScript for QA Engineers

An interactive, self-paced learning app to teach and revise **TypeScript** with a focus on **QA Automation Engineering**.

> 🤖 Built with the help of AI (Claude) as part of my ongoing self-learning journey — demonstrating how I actively use AI tools to upskill as a QA engineer.

---

## ✨ Features

- **10 structured lessons** covering TypeScript from basics to advanced QA patterns
- **Syntax-highlighted code examples** tailored to real QA scenarios (Playwright, Cypress, Jest)
- **Interactive quizzes** after each lesson with instant feedback
- **Progress tracking** — marks lessons as completed as you go
- **Zero setup** — pure HTML/CSS/JS, no build tools or dependencies

---

## 📚 Lessons Covered

| # | Topic | QA Focus |
|---|-------|----------|
| 1 | Why TypeScript for QA | Motivation & tooling benefits |
| 2 | Types & Variables | `string`, `number`, `boolean`, `any` |
| 3 | Interfaces & Types | Modelling API responses & page objects |
| 4 | Functions | Typed test helpers & callbacks |
| 5 | Classes | Page Object Model (POM) pattern |
| 6 | Generics | Reusable test utilities |
| 7 | Enums | Test status, environments, roles |
| 8 | Async / Await | Handling async Playwright actions |
| 9 | Advanced Patterns | Utility types, type guards, union types |
| 10 | Config & Tooling | `tsconfig.json`, ESLint, strict mode |

## 🚀 How to Run

No installation needed. Just open the file in your browser:

```bash
open index.html
```

Or use **VS Code Live Server**:
1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
2. Right-click `index.html` → **Open with Live Server**
